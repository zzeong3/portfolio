  1. 1. PC의 왼쪽 하단 윈도우 버튼 옆에 있는 검색 버튼을 선택 후 "카메라"를 검색하여 선택합니다.

마이크 인식
  1. 1. PC의 왼쪽 하단 윈도우 버튼 옆에 있는 검색 버튼을 선택 후 "소리 설정"을 검색하여 선택합니다.
  2. 2. 마이크 잭 연결이 정상적으로 되었으나 마이크 테스트 시 소리 볼륨바가 좌우로 움직이지 않는다면 마이크가 정상적으로 인식이 안된 상태입니다.
    이런 경우 하단 또는 우측 관련 설정의 "사운드 제어판" 을 선택해 주세요.
  3. 3. 소리 설정 화면이 나타나면 녹음 메뉴의 마이크를 선택 후 우클릭을 하여 나타난 리스트에서 "사용"으로 선택합니다.
  4. 4. 이후 하단 "속성"을 선택 하여 상단 "수준" 메뉴의 "마이크" , "마이크증폭"을 확인해주세요.

오디오 확인
  1. 1. 소리가 정상적으로 작동하지 않는 경우 PC 우측 하단의 볼륨 버튼을 마우스 우클릭하여 "소리 설정 열기"를 선택해 줍니다.
  2. 2. 소리 설정 화면이 나타나면 출력 장치에 사용자의 스피커 또는 헤드셋이 선택해주세요.
돌발 대처법
  1. 1. 영상면접 진행 중 의도치 않게 화면이 종료되었어도 걱정하지 마세요.
    - 단, 나가기 선택 시에는 재입장이 불가합니다.
  2. 2. 영상면접 홈에서 영상면접에 재입장 할 수 있도록 안내해 드립니다
웹캠 작동 확인
  1. 1. PC의 하단 Dock에서 Launch pad를 선택합니다.
  2. 2. 맥북 카메라를 이용하기 위해서는 Face Time 또는 Photo Booth 중 하나를 선택하세요.
  3. 3. 위의 앱 실행 후 맥북 카메라 상단 녹색 불이 들어온다면 카메라가 정상 작동 상태입니다.
  4. 4. 만약 위의 방법으로 진행 시 카메라가 정상 작동되지 않을 경우 시스템 정보보기 (응용 프로그램 폴더 안의 유틸리티 폴더)를 확인하세요.
